This is an example of a table using HTML tags and attributes.

Name Scores
Math Science
Lokender 85 90
Pranav 88 92

The table above demonstrates the use of the following tags:

  • <table>: Defines the table.
  • <tr>: Defines a table row.
  • <th>: Defines a table header cell.
  • <td>: Defines a table data cell.

Attributes used:

  • border: Specifies the border of the table.
  • : Merges cells vertically.
  • : Merges cells horizontally.
