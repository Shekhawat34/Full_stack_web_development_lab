HTML Lists

Understanding HTML Lists

HTML provides two main types of lists: ordered lists and unordered lists. Each type serves a different purpose in organizing content.

Unordered Lists

An unordered list is used to group items without a specific order. It is created using the `

    ` tag, with each item inside an `
  • ` tag.

    Example of Unordered List

    Here is an example of an unordered list:

    <ul>
      <li>Item 1</li>
      <li>Item 2</li>
      <li>Item 3</li>
    </ul>
    • Item 1
    • Item 2
    • Item 3

    Ordered Lists

    An ordered list is used when the order of items matters. It is created using the `

      ` tag, with each item inside an `
    1. ` tag.

      Example of Ordered List

      Here is an example of an ordered list:

      <ol>
        <li>First Item</li>
        <li>Second Item</li>
        <li>Third Item</li>
      </ol>
      1. First Item
      2. Second Item
      3. Third Item

      Nesting Lists

      You can also nest lists within each other to create sub-lists. This can be done by placing a `

        ` or `
          ` inside an `
        1. ` of another list.

          Example of Nested Lists

          Here is an example of a nested list:

          <ul>
            <li>Main Item 1</li>
            <li>Main Item 2<br>
              <ul>
                <li>Sub Item 1</li>
                <li>Sub Item 2</li>
              </ul>
            </li>
          </ul>
          • Main Item 1
          • Main Item 2
            • Sub Item 1
            • Sub Item 2

          Ordered List in Unordered List

          Sometimes, you may want to include an ordered list within an unordered list. This can be useful for categorizing items that have a specific order.

          Example of Ordered List in Unordered List

          Here is an example:

          <ul>
            <li>Category 1</li>
            <li>Category 2<br>
              <ol>
                <li>Ordered Item 1</li>
                <li>Ordered Item 2</li>
              </ol>
            </li>
          </ul>
          • Category 1
          • Category 2
            1. Ordered Item 1
            2. Ordered Item 2

          Definition Lists

          In addition to ordered and unordered lists, HTML also supports definition lists, which are used to define terms and their descriptions. This is done using the `

          `, `
          `, and `
          ` tags.

          Example of Definition List

          Here is an example of a definition list:

          <dl>
            <dt>Term 1</dt>
            <dd>Description for Term 1</dd>
            <dt>Term 2</dt>
            <dd>Description for Term 2</dd>
          </dl>
          Term 1
          Description for Term 1
          Term 2
          Description for Term 2

          Conclusion

          Lists are a fundamental part of HTML that help organize content effectively. By using unordered and ordered lists, you can present information clearly and logically. Nesting lists allows for even more complex structures, making it easy to categorize related items.
