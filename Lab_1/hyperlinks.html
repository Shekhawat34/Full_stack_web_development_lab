HTML Hyperlinks

Understanding HTML Hyperlinks

Hyperlinks are essential for navigating the web. They allow users to click and jump to different pages or sections of a page.

The `` Tag

The `` tag is used to create hyperlinks. The most important attribute of this tag is `href`, which specifies the URL of the page the link goes to.

Example of a Hyperlink

Here is an example of a hyperlink:

<a href="https://www.example.com">Visit Example</a>

This will create a link that says "Visit Example".

Visit Example

Target Attribute

The `target` attribute specifies where to open the linked document. The most common values are:

  • _blank: Opens the link in a new tab or window.
  • _self: Opens the link in the same frame (default behavior).
  • _parent: Opens the link in the parent frame.
  • _top: Opens the link in the full body of the window.

Example with Target Attribute

Here is an example of a hyperlink that opens in a new tab:

<a href="https://www.example.com" target="_blank">Visit Example in New Tab</a>

This will create a link that opens "Visit Example" in a new tab when clicked.

Visit Example in New Tab

Summary

Hyperlinks are created using the `` tag with the `href` attribute to specify the link's destination. The `target` attribute can be used to control how the link opens, enhancing user navigation.
